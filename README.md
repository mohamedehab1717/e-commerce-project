📌 Overview

This project is a full-stack E-commerce web application that allows users to browse products, add items to the cart, and place orders.
It also includes an Admin Panel where administrators can manage products (add, edit, delete).

The project demonstrates building a complete web application using Node.js, Express, MySQL, and Vanilla JavaScript.

 Features :
 User Features:

User registration and login

JWT authentication

Browse available products

Add products to cart

Checkout and place orders

   Admin Features:

Add new products

Edit existing products

Delete products

View all products

   Technologies Used:
Backend:

Node.js

Express.js

MySQL

JSON Web Token (JWT)

bcryptjs

CORS

dotenv

Frontend:

HTML5

CSS3

JavaScript (Vanilla JS)

Tools:

Git & GitHub

Render (Deployment)

VS Code

Postman (API testing)

   Database:

The project uses MySQL with the following main tables:

Users

Products

Orders

Order_Items

  Authentication:
Authentication is implemented using JWT tokens.
Passwords are securely hashed using bcrypt before storing in the database.
